# brainfuck
A brainfuck compiler.

## System requirements
- Python (refer to [pyproject.toml](./pyproject.toml) for the exact version).
- [Poetry](https://python-poetry.org/).
- GCC.

## Basic usage
- `$ poetry install && poetry run brainfuck ./path_to_some_file`
